Detailed Description

Hartman3 unconstrained test problem.

  • Number of parameters: 3
  • Domain: \(\bar{\vec{x}} \in [0, 1]^3\)
  • Optimal point: \(\bar{\vec{x}}_{\text{opt}} = (0.1146398, 0.5556488, 0.8525470)\)
  • Optimal function value: \(\bar{f}(\bar{\vec{x}}_{\text{opt}}) = -3.862785\)

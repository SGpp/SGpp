Computes a string representation of attrValue that is
valid json. Supported are:
1) boolean
2) integer
3) dictionary
4) float
5) list, np.ndarray
6) tuple
7) string
8) DataVector
9) objects with ".toJson" method
10) objects with ".serialize" method
